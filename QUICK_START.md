# Quick Setup Guide

## Installation Steps

1. **Download all files** to your computer
2. **Create a new folder** (e.g., "Seven-Heaven-Spa")
3. **Place all files** in the same folder:
   - index-enhanced.html
   - app-enhanced.js
   - logo.png
   - README_ENHANCED.md (optional - for reference)

4. **Open** `index-enhanced.html` in your web browser
   - Double-click the file, or
   - Right-click and select "Open with" â†’ Your browser

5. **Login** with demo credentials:
   - Admin: username `admin` / password `admin123`
   - Staff: username `sara` / password `staff123`

## That's it! You're ready to go.

## Important Files

### Required Files (Must be in same folder):

## Mobile Access
Works great on mobile browsers too! Just open the HTML file on your phone.

## First Time Setup

### For Admin:
1. Login as admin
2. Go to Staff tab - Add your real staff members
3. Go to Services tab - Review services and pricing
4. Start creating bookings!

### For Staff:
1. Login with your credentials (admin will create these)
2. View your schedule in "My Schedule" tab
3. Confirm payments for completed bookings
4. Log transportation in "Transportation" tab
